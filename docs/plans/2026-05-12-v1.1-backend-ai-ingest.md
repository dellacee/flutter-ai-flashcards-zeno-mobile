# V1.1 Backend + AI Ingest Pipeline — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship một backend FastAPI trên Cloud Run + AI pipeline biến PDF/URL/text thành flashcards tự động lưu vào Firestore. Client Flutter wire upload sheet + status streaming. **Chưa có review screen (Plan 3).**

**Architecture:** Client uploads file/text → backend FastAPI (Cloud Run, asia-southeast1) authenticates qua Firebase ID token → parse content → chunk → LLM gen cards (Gemini Flash) → batch write Firestore `users/{uid}/decks/{deckId}/cards/*`. Status doc tại `users/{uid}/ingests/{ingestId}` cho client subscribe progress. Backend service account có Firestore write quyền.

**Tech Stack:** Python 3.12 + FastAPI + uvicorn + Pydantic v2 + google-cloud-firestore + firebase-admin (token verification) + google-generativeai (Gemini SDK) + pypdf (PDF parse) + httpx (URL fetch) + trafilatura (article extract) + Docker + Cloud Run. Client: Flutter Riverpod + dio for multipart upload.

**Spec reference:** [docs/specs/2026-05-08-ai-learning-companion-design.md](../specs/2026-05-08-ai-learning-companion-design.md)
**Prior plan:** [docs/plans/2026-05-08-v1.0-foundation-and-auth.md](2026-05-08-v1.0-foundation-and-auth.md) — V1.0 shipped at tag `v0.1.0-foundation`.

## Test strategy

- **Backend**: pure pytest, no FastAPI test client side-effects. Use `httpx.AsyncClient` for endpoint integration. Mock `google.generativeai` + Firestore with `pytest-mock` + dataclass fakes.
- **Client**: widget tests for sheet + progress UI with `mocktail` injected `IngestRepository`. Integration of backend ↔ client done manually via emulator + local backend (`uvicorn --reload`).
- **No flaky internet**: every test must be hermetic. Touching Gemini / Firestore in CI is forbidden.

Backend CI runs on `ubuntu-latest`, Python 3.12, `uv` package manager.

```python
import structlog
from fastapi import Depends, Header, HTTPException, status
from firebase_admin import auth as firebase_auth, credentials, initialize_app, get_app
from app.config import get_settings

log = structlog.get_logger(__name__)


class AuthError(Exception):
    pass


def _ensure_firebase_initialized() -> None:
    try:
        get_app()
    except ValueError:
        settings = get_settings()
        if settings.google_application_credentials:
            cred = credentials.Certificate(settings.google_application_credentials)
            initialize_app(cred, {"projectId": settings.firebase_project_id})
        else:
            # Cloud Run environment uses Application Default Credentials
            initialize_app(options={"projectId": settings.firebase_project_id})


def verify_id_token(token: str) -> str:
    _ensure_firebase_initialized()
    try:
        decoded = firebase_auth.verify_id_token(token)
        return decoded["uid"]
    except Exception as e:
        log.warning("verify_id_token failed", error=str(e))
        raise AuthError("Invalid or expired token") from e


def require_uid(authorization: str = Header(None)) -> str:
    if not authorization or not authorization.startswith("Bearer "):
        raise HTTPException(status.HTTP_401_UNAUTHORIZED, detail="Missing Bearer token")
    token = authorization.removeprefix("Bearer ").strip()
    try:
        return verify_id_token(token)
    except AuthError as e:
        raise HTTPException(status.HTTP_401_UNAUTHORIZED, detail=str(e)) from e
```

```python
from dataclasses import dataclass
from typing import BinaryIO

from pypdf import PdfReader


@dataclass(frozen=True)
class ParsedContent:
    text: str
    page_count: int


def parse_pdf(file: BinaryIO) -> ParsedContent:
    reader = PdfReader(file)
    pages_text: list[str] = []
    for page in reader.pages:
        raw = page.extract_text() or ""
        # collapse internal whitespace runs but preserve newlines as soft breaks
        lines = [line.strip() for line in raw.splitlines() if line.strip()]
        pages_text.append("\n".join(lines))
    return ParsedContent(text="\n\n".join(pages_text), page_count=len(reader.pages))
```

```python
def chunk_text(text: str, *, target_tokens: int = 500, overlap_tokens: int = 50) -> list[str]:
    chars_per_token = 4
    target = target_tokens * chars_per_token
    overlap = overlap_tokens * chars_per_token
    if len(text) <= target:
        return [text]
    chunks: list[str] = []
    start = 0
    while start < len(text):
        end = min(start + target, len(text))
        chunks.append(text[start:end])
        if end == len(text):
            break
        start = end - overlap
    return chunks
```

```python
    type: Literal["qa"]
    front: str = Field(min_length=1)
    back: str = Field(min_length=1)


class ClozeCardDraft(BaseModel):
    type: Literal["cloze"]
    text: str = Field(min_length=1)

    @model_validator(mode="after")
    def _has_marker(self) -> "ClozeCardDraft":
        import re

        if not re.search(r"\{\{c\d+::[^}]+\}\}", self.text):
            raise ValueError("text must contain a {{cN::word}} marker")
        return self


class McqCardDraft(BaseModel):
    type: Literal["mcq"]
    question: str = Field(min_length=1)
    options: list[str] = Field(min_length=2, max_length=6)
    correct_index: int = Field(ge=0)

    @model_validator(mode="after")
    def _index_in_range(self) -> "McqCardDraft":
        if self.correct_index >= len(self.options):
            raise ValueError("correct_index out of range")
        return self


CardDraft = RootModel[
    Annotated[
        QaCardDraft | ClozeCardDraft | McqCardDraft,
        Field(discriminator="type"),
    ]
]
```

```python
import json

import google.generativeai as genai
import structlog

from app.prompts.gen_cards import render_prompt

log = structlog.get_logger(__name__)


class LLMError(Exception):
    pass


class LLMClient:
    def __init__(self, api_key: str, model: str = "gemini-flash-latest") -> None:
        genai.configure(api_key=api_key)
        self._model = genai.GenerativeModel(model)

    async def generate_cards(self, *, passage: str, max_cards: int) -> list[dict]:
        system, user = render_prompt(passage=passage, max_cards=max_cards)
        prompt = f"{system}\n\n{user}"
        resp = await self._model.generate_content_async(
            prompt,
            generation_config={"response_mime_type": "application/json", "temperature": 0.4},
        )
        raw = resp.text
        try:
            data = json.loads(raw)
        except json.JSONDecodeError as e:
            log.warning("LLM returned non-JSON", raw=raw[:200])
            raise LLMError("LLM response was not valid JSON") from e
        if not isinstance(data, list):
            raise LLMError("LLM response was not a JSON array")
        return data
```

```python
import structlog
from pydantic import ValidationError

from app.models.card import CardDraft
from app.services.chunker import chunk_text
from app.services.llm_client import LLMClient, LLMError
from app.services.pdf_parser import ParsedContent

log = structlog.get_logger(__name__)


async def generate_cards(
    *,
    parsed: ParsedContent,
    llm: LLMClient,
    max_cards: int,
) -> list[CardDraft]:
    chunks = chunk_text(parsed.text)
    per_chunk = max(2, max_cards // max(1, len(chunks)))

    all_drafts: list[CardDraft] = []
    for i, chunk in enumerate(chunks):
        try:
            raw_cards = await llm.generate_cards(passage=chunk, max_cards=per_chunk)
        except LLMError as e:
            log.warning("chunk gen failed, skipping", chunk_index=i, error=str(e))
            continue

        for raw in raw_cards:
            try:
                all_drafts.append(CardDraft.model_validate(raw))
            except ValidationError as e:
                log.info("invalid card draft, skipping", error=str(e), raw=raw)
        if len(all_drafts) >= max_cards:
            break

    return all_drafts[:max_cards]
```

```python
from datetime import UTC, datetime

from google.cloud import firestore
from google.cloud.firestore_v1.async_client import AsyncClient

from app.models.card import CardDraft, ClozeCardDraft, McqCardDraft, QaCardDraft


def get_firestore_client() -> AsyncClient:
    return firestore.AsyncClient()


def _draft_to_firestore_dict(draft: CardDraft) -> dict:
    now = datetime.now(UTC)
    base = {"createdAt": now, "updatedAt": now}
    inner = draft.root
    if isinstance(inner, QaCardDraft):
        return base | {"type": "qa", "front": inner.front, "back": inner.back}
    if isinstance(inner, ClozeCardDraft):
        return base | {"type": "cloze", "text": inner.text}
    if isinstance(inner, McqCardDraft):
        return base | {
            "type": "mcq",
            "question": inner.question,
            "options": inner.options,
            "correctIndex": inner.correct_index,
        }
    raise ValueError(f"Unknown draft type: {type(inner)}")  # pragma: no cover


async def write_cards_batch(
    *,
    client: AsyncClient,
    uid: str,
    deck_id: str,
    drafts: list[CardDraft],
) -> list[str]:
    """Write all drafts to users/{uid}/decks/{deck_id}/cards in a single batch,
    incrementing the deck's cardCount.

    Returns the list of created card ids.
    """
    if not drafts:
        return []

    deck_ref = client.collection("users").document(uid).collection("decks").document(deck_id)
    cards_ref = deck_ref.collection("cards")

    batch = client.batch()
    card_ids: list[str] = []
    for draft in drafts:
        new_ref = cards_ref.document()
        batch.set(new_ref, _draft_to_firestore_dict(draft))
        card_ids.append(new_ref.id)

    batch.update(
        deck_ref,
        {"cardCount": firestore.Increment(len(drafts)), "updatedAt": datetime.now(UTC)},
    )
    await batch.commit()
    return card_ids
```

```python
from datetime import UTC, datetime
from enum import Enum
from uuid import uuid4

from google.cloud.firestore_v1.async_client import AsyncClient


class IngestState(str, Enum):
    QUEUED = "queued"
    PARSING = "parsing"
    GENERATING = "generating"
    WRITING = "writing"
    DONE = "done"
    FAILED = "failed"


def _ingest_ref(client: AsyncClient, uid: str, ingest_id: str):
    return (
        client.collection("users")
        .document(uid)
        .collection("ingests")
        .document(ingest_id)
    )


async def create_status(
    *,
    client: AsyncClient,
    uid: str,
    deck_id: str,
    source: str,
    source_label: str,
) -> str:
    ingest_id = uuid4().hex
    now = datetime.now(UTC)
    await _ingest_ref(client, uid, ingest_id).set(
        {
            "deckId": deck_id,
            "source": source,
            "sourceLabel": source_label,
            "state": IngestState.QUEUED.value,
            "cardsGenerated": 0,
            "createdAt": now,
            "updatedAt": now,
        }
    )
    return ingest_id


async def update_state(
    *, client: AsyncClient, uid: str, ingest_id: str, state: IngestState
) -> None:
    await _ingest_ref(client, uid, ingest_id).update(
        {"state": state.value, "updatedAt": datetime.now(UTC)}
    )


async def mark_done(
    *, client: AsyncClient, uid: str, ingest_id: str, cards_generated: int
) -> None:
    await _ingest_ref(client, uid, ingest_id).update(
        {
            "state": IngestState.DONE.value,
            "cardsGenerated": cards_generated,
            "updatedAt": datetime.now(UTC),
        }
    )


async def mark_failed(
    *, client: AsyncClient, uid: str, ingest_id: str, error: str
) -> None:
    await _ingest_ref(client, uid, ingest_id).update(
        {"state": IngestState.FAILED.value, "error": error, "updatedAt": datetime.now(UTC)}
    )
```

```python
import structlog
from fastapi import APIRouter, BackgroundTasks, Depends, HTTPException, status

from app.auth import require_uid
from app.config import Settings, get_settings
from app.models.ingest import IngestStartedResponse, IngestTextRequest
from app.services.card_generator import generate_cards
from app.services.firestore_client import get_firestore_client, write_cards_batch
from app.services.ingest_status import (
    IngestState,
    create_status,
    mark_done,
    mark_failed,
    update_state,
)
from app.services.llm_client import LLMClient
from app.services.pdf_parser import ParsedContent

log = structlog.get_logger(__name__)
router = APIRouter(prefix="/ingest", tags=["ingest"])


async def _process_text(
    *,
    uid: str,
    ingest_id: str,
    deck_id: str,
    text: str,
    max_cards: int,
    settings: Settings,
) -> None:
    fs_client = get_firestore_client()
    try:
        await update_state(
            client=fs_client, uid=uid, ingest_id=ingest_id, state=IngestState.GENERATING
        )
        llm = LLMClient(api_key=settings.gemini_api_key)
        parsed = ParsedContent(text=text, page_count=1)
        drafts = await generate_cards(parsed=parsed, llm=llm, max_cards=max_cards)

        await update_state(
            client=fs_client, uid=uid, ingest_id=ingest_id, state=IngestState.WRITING
        )
        await write_cards_batch(client=fs_client, uid=uid, deck_id=deck_id, drafts=drafts)

        await mark_done(
            client=fs_client, uid=uid, ingest_id=ingest_id, cards_generated=len(drafts)
        )
    except Exception as e:
        log.exception("ingest failed", uid=uid, ingest_id=ingest_id)
        await mark_failed(client=fs_client, uid=uid, ingest_id=ingest_id, error=str(e))

        uid=uid,
        ingest_id=ingest_id,
        deck_id=req.deck_id,
        text=req.text,
        max_cards=req.max_cards,
        settings=settings,
    )
```

```python
@router.post("/pdf", status_code=status.HTTP_202_ACCEPTED, response_model=IngestStartedResponse)
async def ingest_pdf(
    deck_id: str = Form(..., alias="deckId"),
    max_cards: int = Form(20, alias="maxCards", ge=1, le=50),
    file: UploadFile = File(...),
    uid: str = Depends(require_uid),
    settings: Settings = Depends(get_settings),
    background: BackgroundTasks = ...,
) -> IngestStartedResponse:
    if file.size and file.size > settings.max_pdf_mb * 1024 * 1024:
```

## Reference & gotchas

- **Cloud Run cold start** ~3-5s with our image size. For dev, acceptable. If users complain → enable min-instances=1 (~$5/month).
- **Gemini Flash rate limit** 1500 RPD free, 10 RPS. With ~3 chunks/PDF avg → 500 PDFs/day free.
- **Firestore writes**: each card = 1 write op. 20 cards + 1 deck update = 21 writes. Free tier 20K/day = ~950 ingests/day before paying. Plenty.
- **Cloud Run timeout**: max 60min for HTTP request, but our handler returns 202 in <1s — actual work in background task. Background tasks run as long as the instance is alive; Cloud Run kills idle instances after 15min of no traffic. For long PDFs, this could cut off. Mitigate by capping `max_cards_per_ingest` so total work <30s typically.
- **Service account JSON**: NEVER commit. Use Secret Manager for prod (already in deploy script). For local dev, mount the JSON via `GOOGLE_APPLICATION_CREDENTIALS` env var pointing to a gitignored file.
- **Emulator + localhost**: Android emulator reaches host via `10.0.2.2`, not `localhost`. Use `--dart-define=BACKEND_URL=http://10.0.2.2:8080` when developing against local backend.
- **CORS**: Allow your debug origins (`http://localhost:*`) but tighten to specific Cloud Run URL in production.
